## UIのこだわり
- 軽快さ
  - SPA特有の高速描画
- 誰もが覚えやすいUI
  - タイトルは2つ以上作らず定位置
  - 真ん中にメインコンテンツ
  - 左のサイドメニューは各機能への導線
  - 右のサイドメニューは固有のUI（スレ検索フォーム、書き込み欄など）
- レスポンシブデザイン
  - PC版：デフォルトでメニューを開いた状態
